# Evaluation Outputs

This folder contains a copy of the local evaluation artifacts generated by the study agent.

## Files

- `benchmark_cases.json`: 8 benchmark cases covering factual retrieval, cross-modal retrieval, analytical/multi-hop synthesis, and personalised follow-up.
- `evaluation_results.json`: Full per-case, per-mode evaluation output.
- `evaluation_summary.csv`: Mean metrics for each ablation mode.
- `failure_cases.md`: Automatically extracted final-agent failure cases.

## Current Summary

The copied summary currently contains these modes:

- `plain_llm`
- `text_only`
- `caption_only`
- `no_visual`
- `no_router`
- `no_rerank`
- `no_memory`
- `final_agent`

Metrics include:

- `mean_recall_at_k`
- `mean_page_recall_at_k`
- `mean_mrr`
- `mean_answer_success_proxy`
- `mean_latency_ms`

These files are copied from `sourcecode/data/evaluation/`.
